MAG CAMP — الإصدار 7.1.0 (استقرار النشر)
تاريخ الإصدار: 2026-07-20

تم البناء على نفس مشروع المرحلة السادسة ونفس قاعدة البيانات دون حذف أو إعادة إنشاء البيانات.

التعديلات:
1. تثبيت Python 3.12.11 في .python-version وrender.yaml.
2. تثبيت إصدارات حزم التشغيل، وإضافة Werkzeug صراحة لمنع اختلافات بيئة البناء.
3. استخدام python -m pip أثناء البناء لضمان التثبيت داخل بيئة Render الصحيحة.
4. إضافة wsgi.py كنقطة تشغيل مستقرة، وتحديث Procfile وRender لاستخدام wsgi:app.
5. إضافة سجلات الوصول والأخطاء إلى Gunicorn لتسهيل تشخيص أي مشكلة لاحقة.
6. تحديث /health ليعرض إصدار النظام، إصدار Python، سلامة قاعدة البيانات، وعدد السجلات الأساسية.
7. تحديث هوية الواجهة إلى Enterprise 7.1.
8. إضافة pytest.ini حتى تعمل الاختبارات من جذر المشروع مباشرة.

طريقة النشر:
- ارفع محتويات الملف إلى جذر مستودع GitHub.
- في Render اختر Manual Deploy ثم Clear build cache & deploy.
- يجب أن يعرض /health الحقل version بالقيمة 7.1.0.
