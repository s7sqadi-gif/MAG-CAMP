# نشر MAG CAMP كرابط تجريبي على Render

1. أنشئ حساباً في GitHub وحساباً في Render.
2. ارفع محتويات هذا المجلد إلى مستودع GitHub جديد.
3. في Render اختر **New > Blueprint**.
4. اربط المستودع، وسيقرأ Render ملف `render.yaml` تلقائياً.
5. اضغط Deploy وانتظر حتى تظهر حالة **Live**.
6. ستحصل على رابط عام يعمل من الجوال والكمبيوتر.
