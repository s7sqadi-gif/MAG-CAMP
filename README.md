# MAG CAMP — Phase 4 Foundation v4.0 Beta

هذه النسخة مبنية مباشرة فوق المرحلة الثالثة دون حذف البيانات أو إعادة بناء النظام.

## المنجز في المرحلة الرابعة
- تفعيل حساب **أمير سهيل** كـمراقب سكن.
  - الرقم الوظيفي: `96880`
  - كلمة المرور المؤقتة: `123456`
  - يجب تغييرها عند أول تسجيل دخول.
- استبدال **جرد دورات المياه** بـ **بلاغات دورات المياه**.
- كل بلاغ دورة مياه ينشئ تلقائيًا تذكرة صيانة مرتبطة.
- المشرف والمراقب يشاهدان البلاغات التي قاما بإنشائها فقط.
- المستخدمون وسجل العمليات متاحان للإدارة فقط.
- تقييد بلاغات الصيانة للمشرفين والمراقبين على البلاغات الخاصة بهم.
- إضافة نظام طلبات **إضافة عامل جديد** و**حذف/أرشفة عامل** للمشرفين.
- الطلب لا ينفذ مباشرة؛ يعتمد وينفذ فقط بواسطة مدير السكن أو مدير الخدمات المساندة.
- المشرف يستطيع إنشاء الطلب فقط للعامل أو الغرفة الواقعة ضمن نطاقه.
- تحديث لوحة التحكم حتى تعرض للمشرف بياناته وطلباته وبلاغاته فقط.
- المسار `/health` يعيد `phase: 4`.

## النشر
ارفع محتويات المجلد إلى مستودع GitHub المرتبط بـ Render. قاعدة البيانات داخل `data/mhoms.db` تحتوي على بيانات المرحلة الثالثة مع إضافات المرحلة الرابعة.
