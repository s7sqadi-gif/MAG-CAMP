# MAG CAMP — Phase 4.1 Beta

الإصدار الثاني من المرحلة الرابعة لنظام إدارة سكن ولي العهد.

## التحديثات
- عرض جميع الغرف التابعة للمشرف في شاشة الجولة الأسبوعية.
- عرض أسماء العمال وأرقامهم داخل كل غرفة قبل تنفيذ الجولة.
- إظهار حالة الجولة لهذا الأسبوع: تمت / لم تتم.
- إضافة قسم صيانة داخل جولة الغرفة.
- إنشاء بلاغ صيانة مستقل تلقائياً عند تسجيل عطل من الجولة، مع ربطه بسجل الجولة.
- الإبقاء على نظام بلاغات الصيانة المستقل كما هو.
- مشرف السكن يرى غرفه وجولاته وبلاغاته فقط.
- مشرف الصيانة يرى البلاغات المسندة إليه والبلاغات الجديدة غير المسندة.
- مدير الصيانة ومدير السكن يريان جميع البلاغات، مع إمكانية إسناد البلاغ لمستخدم الصيانة.
- المحافظة على تفعيل حساب أمير سهيل وبقية صلاحيات Phase 4.0.

## التشغيل
```bash
pip install -r requirements.txt
python app.py
```
