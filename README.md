# MAG CAMP Cloud – Phase 1

نسخة سحابية تجريبية جاهزة للنشر، تشمل:
- تسجيل دخول عربي/إنجليزي
- قاعدة بيانات العمال والغرف
- المستخدمين والصلاحيات
- البحث
- طلب تسكين جديد
- طلب نقل عامل
- اعتماد الطلبات
- سجل النشاط

python app.py
