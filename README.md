# MAG CAMP — Phase 3 Foundation v3.0 Beta

هذه أول نسخة تنفيذية من المرحلة الثالثة، مبنية مباشرة فوق المرحلة الثانية دون حذف وظائفها.

## المنجز في هذه الحزمة
- تصميم Responsive يعمل على الكمبيوتر والجوال من نفس الكود.
- لوحة تحكم تنفيذية للإشغال، الشواغر، التكدس، الجولات، الطلبات والصيانة.
- فصل استخدام الغرفة عن بيانات العامل.
- استخدامات الغرف: سكن، مستودع، حراسات، مقاول، إدارة، صيانة، مغسلة، مغلق، خارج الخدمة، أخرى.
- حفظ أبعاد الغرف والمساحة والسعة المعتمدة.
- صفحة موحدة لكل غرفة مع المقيمين وسجل تغيير الاستخدام.
- استيراد أحدث ملف تسكين بتاريخ 19-07-2026.
- نسخة احتياطية من قاعدة المرحلة الثانية داخل data/mhoms_phase2_backup.db.
